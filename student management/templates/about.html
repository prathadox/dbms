About Student Management System

Project Overview

The Student Management System is a comprehensive web-based application designed to manage student records, attendance, and academic departments efficiently. This system demonstrates the implementation of CRUD (Create, Read, Update, Delete) operations with a MySQL database backend.

Key Features
  • Complete CRUD operations for student records
  • Department/Branch management
  • Attendance tracking system
  • User authentication and authorization
  • Database triggers for audit logging
  • Search functionality by roll number
  • Responsive Bootstrap UI
Technology Stack
  • Backend: Python 3.14, Flask 3.1.2
  • Database: MySQL/MariaDB
  • ORM: SQLAlchemy 2.0.44
  • Authentication: Flask-Login
  • Frontend: HTML5, CSS3, Bootstrap 5
  • JavaScript: jQuery, AOS, Owl Carousel
Educational Purpose

This project is developed as a DBMS Mini-Project to demonstrate practical knowledge of database management, web development, and full-stack application architecture. It showcases various database concepts including:

  • Database design and normalization
  • SQL triggers and stored procedures
  • CRUD operations and data validation
  • User authentication and session management
  • MVC architecture pattern
